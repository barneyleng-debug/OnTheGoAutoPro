OnTheGo Auto Pro — website package

Files:
- index.html       : main webpage
- styles.css       : styling
- script.js        : map + form JS
- hero-van.jpg     : placeholder image (optional) - not included
- README.txt       : this file

How to use:
1. Unzip the package and upload files to your web host (Netlify, Vercel, Bluehost, your cPanel, etc.)
2. If you don't have hosting:
   - Free options: Netlify, Vercel (drag-and-drop), GitHub Pages.
3. The booking form uses a mailto: fallback. For a proper form backend:
   - Use Formspree, Getform, or configure your server to accept POST submissions.
4. Replace social media links and hero-van.jpg with your actual logo and photos.
5. To edit service area or phone, open index.html and modify the window.__OGAP_CONFIG block in the page or edit script.js.
